## 项目概述

这是一个完全运行在 Windows 平台的图形化塔防游戏，玩家通过种植植物抵御不断来袭的僵尸，最终目标是累计击杀 40 只僵尸。

**开发历程**：
- ✅ 完整的用户系统（登录/注册/忘记密码）
- ✅ 3种特色植物（豌豆射手、向日葵、大嘴花）
- ✅ 动态僵尸波次（速度/间隔随进度变化）
- ✅ 阳光收集与铲子系统
- ✅ 开场动画 + 胜利/失败判定
- ✅ 全键盘/鼠标交互

**代码规模**：约 3100 行 C++ 代码，纯面向对象设计。

## 🎮 游戏特性

### 🌱 植物系统
| 植物 | 阳光消耗 | 生命值 | 技能 |
|------|---------|--------|------|
| 豌豆射手 | 100 | 150 | 直线发射豌豆（20伤害） |
| 向日葵 | 50 | 100 | 每隔 2000 帧生产 25 阳光 |
| 大嘴花 | 150 | 200 | 近战吞噬僵尸（需咀嚼 20 秒） |

### 🧟 僵尸系统
- 普通僵尸：血量 100，攻击力 10
- 动态波次：随击杀数增加生成速度和移动速度
  - 第1只 → 间隔 5000ms，速度 0.15
  - 前5% → 间隔 1500ms，速度 0.20
  - 前10% → 间隔 1000ms，速度 0.25
  - 前25% → 间隔 500ms，速度 0.25
  - 前50% → 间隔 100ms，速度 0.30
  - 前75% → 间隔 50ms，速度 0.35
  - 最后25% → 间隔 10ms，速度 0.40

### ☀️ 阳光系统
- 自然掉落：每隔 1000-1500 帧掉落一个阳光
- 向日葵生产：每 2000 帧生产一个阳光
- 点击收集：阳光飞向计数区，增加 25 阳光

### 🛠️ 铲子功能
- 点击铲子卡片激活，图标跟随鼠标
- 左键点击植物 → 删除植物
- 右键点击 → 取消铲子状态

### 🏆 胜利/失败条件
- 胜利：累计击杀 40 只僵尸
- 失败：有僵尸到达房子（x < 56）
- 结束后显示对应图片，播放音效，停留 3 秒后退出

### 🎬 开场动画
- 背景从右向左滚动
- 11 只僵尸呈 > 形排列
- 卡片栏从上向下滑入

## 🚀 快速开始

### 环境准备
1. 安装 Visual Studio 2022
2. 安装 EasyX 图形库：https://easyx.cn

### 获取项目
git clone https://github.com/hls-9527/Plants-Vs-Zombies.git

或者直接下载 ZIP 解压。

### 编译运行
1. 双击 Plants-Vs-Zombies.sln 打开项目
2. 确保项目配置为 Release/x64
3. 按 F5 编译运行

### 资源文件
所有图片和音效已放置在 src/resources/ 目录下，无需额外配置。

## 🎯 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标左键 | 点击卡片选择植物/铲子，点击网格种植，点击阳光收集 |
| 鼠标右键 | 取消选中植物/铲子 |
| 鼠标移动 | 选中植物/铲子时，图标跟随鼠标 |
| ESC键 | 在开始菜单退出游戏 |
