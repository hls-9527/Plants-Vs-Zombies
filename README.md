# 🧟‍♂️ 植物大战僵尸 - EasyX复刻版

欢迎来到 **植物大战僵尸 EasyX复刻版** 项目！这是一个基于C++和EasyX图形库开发的塔防游戏，完整复刻了经典游戏的核心玩法。玩家通过种植不同功能的植物，抵御一波波来袭的僵尸，最终目标是累计击杀40只僵尸取得胜利。

## 项目概述

本项目是一个完全运行在Windows平台的图形化塔防游戏，从零实现图形渲染、碰撞检测、动画系统和游戏逻辑，完整还原经典游戏的核心体验。

**主要特点**：
- ✅ **完整的游戏流程**：开始菜单 → 开场动画 → 游戏主循环 → 胜利/失败判定
- ✅ **三种特色植物**：豌豆射手（远程攻击）、向日葵（生产阳光）、大嘴花（近战吞噬）
- ✅ **动态僵尸波次**：随击杀数增加，僵尸生成速度和移动速度逐渐加快
- ✅ **阳光经济系统**：自然掉落 + 向日葵生产，点击收集阳光用于种植
- ✅ **铲子工具**：可随时铲除已种植的植物
- ✅ **完整的音效与动画**：背景音乐、音效、植物/僵尸动画、开场动画

**代码规模**：约 3100 行 C++ 代码，纯面向对象设计。

## 🎮 游戏特性

### 🌱 植物系统
| 植物 | 阳光消耗 | 生命值 | 技能说明 |
|:---|:---|:---|:---|
| **豌豆射手** | 100 | 150 | 直线发射豌豆子弹，每颗造成20伤害 |
| **向日葵** | 50 | 100 | 每隔2000帧生产25阳光 |
| **大嘴花** | 150 | 200 | 近战吞噬前方僵尸，需咀嚼2000帧恢复 |

### 🧟 僵尸系统
- **普通僵尸**：血量100，攻击力10，移动速度随波次动态变化
- **动态波次机制**（击杀数越多，僵尸越强）：

| 击杀进度 | 生成间隔 | 移动速度 |
|:---|:---|:---|
| 第1只 | 5000帧 | 0.15 |
| 前5% | 1500帧 | 0.20 |
| 前10% | 1000帧 | 0.25 |
| 前25% | 500帧 | 0.25 |
| 前50% | 100帧 | 0.30 |
| 前75% | 50帧 | 0.35 |
| 最后25% | 10帧 | 0.40 |

### ☀️ 阳光系统
- **自然掉落**：每隔1000-1500帧从屏幕顶部随机位置掉落一个阳光
- **向日葵生产**：每2000帧从向日葵位置产生一个阳光，向右上方飘起后落回地面
- **点击收集**：鼠标点击阳光即可收集，阳光飞向计数区（162, 0），增加25阳光
- **地面停留**：阳光落地后停留500帧，超时自动消失

### 🛠️ 铲子系统
- **激活**：点击铲子卡片（第9个卡片位），铲子图标跟随鼠标
- **删除植物**：左键点击已种植的植物，将其删除并播放音效
- **取消**：左键点击空地 或 右键点击任意位置，铲子归位

### 🏆 胜利/失败条件
- **胜利条件**：累计击杀40只僵尸，播放胜利画面和音效
- **失败条件**：任何僵尸到达房子（x < 56），播放失败画面和音效
- **结束流程**：画面停留10秒后自动退出

### 🎬 开场动画
游戏开始前播放完整的开场动画：
1. **背景滚动**：背景图片从右向左平滑滚动
2. **僵尸列阵**：11只僵尸呈 `>` 形排列，播放站立动画
3. **卡片下落**：工具栏从屏幕顶部滑入

## 🚀 快速开始

### 环境准备
1. 安装 **Visual Studio 2022**（或更高版本）
2. 安装 **EasyX 图形库**：前往 [https://easyx.cn](https://easyx.cn) 下载并安装

### 获取项目
```bash
git clone https://github.com/hls-9527/Plants-Vs-Zombies.git
```
或者直接下载ZIP压缩包解压。

### 编译运行
1. 双击 `Plants-Vs-Zombies.sln` 打开项目
2. 确保解决方案配置为 `Release`，平台为 `x64`
3. 按 `F5` 编译并运行

### 资源文件
所有图片和音效已放置在 `src/resources/` 目录下，无需额外配置。

## 🎯 操作说明

| 操作 | 功能 |
|:---|:---|
| **鼠标左键** | 点击卡片选择植物/铲子；点击网格种植植物；点击阳光收集 |
| **鼠标右键** | 取消当前选中的植物/铲子 |
| **鼠标移动** | 选中植物/铲子时，图标跟随鼠标 |
| **ESC键** | 在开始菜单界面退出游戏 |

## 📂 核心类结构

项目采用纯面向对象设计，核心类继承体系如下：

```
Plants-Vs-Zombies/
├── src/
│   ├── entities/                           # 实体类
│   │   ├── GameObject.h/cpp                # 🧬 抽象基类（坐标、存活、动画帧）
│   │   ├── Plant/                          # 🌱 植物体系
│   │   │   ├── Plant.h/cpp                 # 植物基类
│   │   │   ├── Peashooter.h/cpp            # 豌豆射手
│   │   │   ├── Sunflower.h/cpp             # 向日葵
│   │   │   └── Chomper.h/cpp               # 大嘴花
│   │   ├── Zombie/                          # 🧟 僵尸体系
│   │   │   ├── Zombie.h/cpp                 # 僵尸基类
│   │   │   └── BasicZombie.h/cpp            # 普通僵尸
│   │   ├── Bullet/                          # 💥 子弹体系
│   │   │   ├── Bullet.h/cpp                 # 子弹基类
│   │   │   └── PeaBullet.h/cpp              # 豌豆子弹
│   │   └── Sunshine.h/cpp                    # ☀️ 阳光实体
│   ├── game/
│   │   └── GameManager.h/cpp                 # 🎮 核心管理器（单例）
│   ├── ui/                                   # 🖥️ 界面类
│   │   ├── StartMenu.h/cpp                   # 开始菜单
│   │   ├── GameUI.h/cpp                      # 游戏界面（背景、卡片）
│   │   └── Shovel.h/cpp                       # 🛠️ 铲子工具
│   ├── utils/                                # 🔧 工具类
│   │   ├── EasyXHelper.h/cpp                  # 透明贴图、帧率控制
│   │   └── AudioManager.h/cpp                 # 音频管理
│   ├── resources/                             # 📁 资源文件
│   │   ├── images/                            # 所有图片
│   │   └── sounds/                            # 所有音效
│   └── main.cpp                               # 🚪 程序入口
├── Plants-Vs-Zombies.sln                      # Visual Studio解决方案
└── README.md                                   # 本文件
```

### 关键类说明
- **`GameObject`**：所有游戏实体的抽象基类，定义 `update()` 和 `draw()` 纯虚接口
- **`GameManager`**：游戏核心，采用单例模式，管理所有对象容器、碰撞检测、游戏状态
- **`EasyXHelper`**：封装透明贴图算法和帧率控制，所有绘图操作均通过此类
- **`AudioManager`**：静态类，统一管理背景音乐和音效的播放与切换
